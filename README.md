# 使い方（OS Xのみ）

1. このリポジトリをダウンロードします。
1. Terminal.appでダウンロードしたフォルダを開きます。
1. `python -m SimpleHTTPServer`と打ち込みます。
1. Webサーバが立ち上がっているので、打ち込んだMacのIPアドレスを調べてiPhoneのSafariでアクセスします。
1. ＋ボタンを押して、ホーム画面に追加を行います。
1. できました〜！
